# 《AI主权》书稿导航

本书采用“书前内容 + 三个部分 + 结语 + 附录”的独立文件结构，共8章。每一章都是独立文件夹，不保留整章合稿。

| 顺序 | 目录 | 内容 |
|------|------|------|
| 00 | `00_书前` | 封面、内容简介、读者承诺、阅读说明、目录、前言 |
| 01 | `01_第一部分_AI主权的觉醒` | 第一至二章：核心定义与AgenticOps方法 |
| 02 | `02_第二部分_四级主权实践` | 第三至六章：个人、组织、企业、政府与国家 |
| 03 | `03_第三部分_安全协同与未来` | 第七至八章：安全规则、全球竞争、协同网络与未来趋势 |
| 04 | `04_结语` | 全书结语 |
| 05 | `05_附录` | 核心术语与参考文献 |
| assets | `assets/images` | 全书图表资源 |

## 章节文件夹结构

每个章节文件夹统一采用以下顺序：

| 文件 | 用途 |
|------|------|
| `00_章节导读.md` | 章标题、开篇内容和小节导航 |
| `01_...md` 起 | 按原章节顺序拆分的正文小节 |
| `90_本章核心要点.md` | 本章结论摘要 |
| `91_本章决策清单.md` | 本章行动与检查问题 |
| `99_本章参考文献.md` | 本章脚注定义和资料来源 |

编辑时先进入对应部分，再进入章节文件夹，按序号编辑。脚注引用保留在正文小节中，定义统一放在该章的 `99_本章参考文献.md`。
